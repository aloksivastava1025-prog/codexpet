// how it works

    
Built on hash, roast, and shame

    
Three steps. Your pet watches, judges, and fixes your code.

    

      

        #
        
Hatch on the web

        
Enter your username, customize your pet's species and hat. The dashboard generates a unique token and a CLI command you paste into your terminal.

      

      

        >_
        
Run the CLI daemon

        
The Python CLI watches your codebase in real-time using watchdog. Every file save triggers an AI analysis through your secure backend. No API keys exposed locally.

      

      

        !!!
        
Get roasted. Get fixed.

        
Your pet roasts your code, plays meme sounds (bruh, womp, emotional damage), and offers the corrected version. Press Y to auto-apply the fix. Earn or lose XP.
